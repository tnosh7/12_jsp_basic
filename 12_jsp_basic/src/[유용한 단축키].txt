[ 유용한 단축키]

1) 메서드 정의부 <> 호출부 이동
 
   - ctrl + 호출메서드 클릭   > 메서드의 정의부로 이동한다.
   (호출부에서 정의부로 이동한 뒤) 
   - 정의부 메서드에서 alt +왼쪽방향키 > 호출부로 이동 
   - 호출부 메서드에서 alt +오른쪽방향키 > 정의부로 이동 


2) 특정 라인으로 이동 (같은 파일)
   - ctrl + l을 이용하여 특정 라인으로 이동할 수 있다.


3) 특정단어 찾기 (같은 파일)
   - ctrl + f를 이용하여 특정단어가 들어간 파일들을 빠르게 검색할 수 있다.
     forward > 아래방향으로 탐색
     backward > 윗방향으로 탐색

    replace with > 탐색후 단어 변경
    scope all > 탐색된 단어를 모두 수정
    selected lines > 마우스 블록을 잡은 영역의 단어만 수정
  

3) 단어 찾기(이클립스 전체)
   3-1) ctrl + h > file search를 통하여 특정 단어가 들어간 파일을 모두 검색할 수 있다.
   3-2) 단어 블록 + ctlr+ alt + g > 블록된 단어가 들어간 파일을 모두 검색할 수 있다.


4) 파일 찾기(이클립스 전체)
   ctrl + shift + r > 파일의 이름에 특정단어가 들어간 파일을 모두 검색할 수 있다.
